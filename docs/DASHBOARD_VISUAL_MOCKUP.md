## Interaction Behaviors

### Robot Cards
- **Hover**: Border changes from gray-700 to primary (#58a6ff)
- **Click**: Navigate to robot detail page
- **Repair Button**: Only shows if HP < maxHP
- **Portrait**: Placeholder until image system implemented

### HP Bars
- **Transition**: Smooth width transition (300ms)
- **Color**: Changes instantly based on percentage thresholds
- **Percentage**: Displayed on right with matching color

### Quick Actions
- **Primary Button**: Solid fill with hover darken
- **Secondary Button**: Outline with hover fill
- **Disabled Button**: No hover, cursor not-allowed
