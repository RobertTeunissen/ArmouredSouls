## Purpose

- Proposed balance modifications
- Economic adjustments to weapons and combat
- Implementation details for balance changes
- Analysis of balance impact

## Balance Philosophy

Balance changes in Armoured Souls aim to:
- Maintain viable diversity in robot builds
- Ensure no single strategy dominates
- Create meaningful choices in weapon selection
- Balance risk/reward across different playstyles
- Support economic sustainability across all archetypes

## Usage

When proposing or implementing balance changes:
1. Document the problem being addressed
2. Analyze impact on different playstyles
3. Consider economic implications
4. Test with diverse robot builds
5. Update related PRDs and formulas
6. Document the final implementation
