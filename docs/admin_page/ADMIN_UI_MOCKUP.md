## Responsive Behavior

### Desktop (> 1024px):
- Modal: Max width 6xl (1152px)
- Attribute grid: 4 columns
- Battle table: All columns visible

### Tablet (768px - 1024px):
- Modal: Max width full with padding
- Attribute grid: 3 columns
- Battle table: Scroll horizontally

### Mobile (< 768px):
- Modal: Full width, max height 90vh
- Attribute grid: 2 columns
- Battle table: Cards instead of table
- Simplified view for combat log

## Interaction States

### Buttons:
- **Default**: bg-blue-600, hover:bg-blue-700
- **Disabled**: bg-gray-600, cursor-not-allowed
- **Loading**: bg-gray-600, "Loading..." text

### Combat Events:
- **Collapsed**: Shows icon, timestamp, message
- **Expanded**: Shows formula breakdown in green monospace
- **Click anywhere**: Toggle expand/collapse

### Search/Filter:
- **Input focus**: Border highlight
- **Enter key**: Triggers search
- **Empty results**: Shows helpful message

## Accessibility

- Semantic HTML (table, button, input elements)
- Keyboard navigation supported
- Click handlers on appropriate elements
- Color coding supplemented with icons
- Clear visual hierarchy
- Readable font sizes (text-sm, text-base)
- High contrast text on dark background

## Performance Notes

- Pagination limits data transfer (20-100 battles per page)
- Battle details loaded on-demand (not preloaded)
- Virtual scrolling not needed (reasonable list sizes)
- Formula breakdowns rendered conditionally
- No animation/transitions (fast rendering)
