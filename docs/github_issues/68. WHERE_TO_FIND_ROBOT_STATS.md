## Troubleshooting "I Don't See It"

### If you don't see the Robot Statistics section:

1. **Check you're on the Admin page**
   - URL should be: `http://localhost:5173/admin`
   - Navigation should show "Admin" as active/highlighted

2. **Scroll down the page**
   - The section is in the middle/lower part of the page
   - Below "Bulk Cycle Testing"
   - Above "Battle Logs & Debugging"

3. **Check browser zoom**
   - Make sure zoom is at 100%
   - Try Ctrl+0 (or Cmd+0 on Mac) to reset zoom

4. **Clear cache and reload**
   - Press Ctrl+Shift+R (or Cmd+Shift+R on Mac)
   - This ensures you have the latest version

5. **Verify you're logged in as admin**
   - Regular users won't see the Admin page
   - You need admin role in the database

6. **Check frontend is running**
   - Terminal should show "VITE" server running
   - Port should be accessible
